Lawn Buddy



What to Do

Welcome to the Lawn Buddy Web App! The idea behind the app is to make life easier for the homeowner looking to take care of their yard. The inspiration behind this project was to create a system that would be able to get weather information in my area as I needed it. the web application is connected to a weather API that will update what's displayed on the screen when refreshed.

What to Do

All you need to do is click the button that you want information and the web page will adjust accordingly. The app should be pretty straight forward with little to do but click here and there.
